Razor 页面文件

Razor Syntax

Razor 内容页面充当生成 HTML 的模板。 典型的内容页面包括静态 HTML、动态发出 HTML 的标签助手和 C# 代码。 C# 代码根据一组规则或语法嵌入到静态内容和标签助手中。

所有代码块都必须出现在 @{ ... } 括号内。 只要您键入 @，就假定您已经开始编写 C# 代码。 下面的所有内容都被假定为代码，除非你告诉 Razor 否则： 

@{
    var numbers = Enumerable.Range(1, 10); //Get numbers from 1 - 10
    foreach(var number in numbers){
        ...
   }
}

变量的值通过在它们前面加上 @ 符号来呈现到 HTML 中： 

The time is @DateTime.Now

如果要在上述示例中的循环中呈现变量编号，则必须在它前面加上 @ 符号：

@{
    var numbers = Enumerable.Range(1, 10); //Get numbers from 1 - 10
    foreach(var number in numbers){
           @number
    }
}

更常见的是，您会看到在与 Razor 代码分开的代码块中声明的语句，用于呈现它们。 迭代和选择语句关键字（if、for、foreach、switch、while 等）在用于控制渲染时不需要在它们之前使用大括号。 一个简单的 @ 符号就可以了。 对于 try...catch 也是如此，如果您想内联使用该特定构造： 

@{ var numbers = Enumerable.Range(1, 10); //}
   
@foreach(var number in numbers){
   @number
}

如果要呈现单行表达式的结果，请使用 @( ... ) 语法： 

@{var numbers = Enumerable.Range(1, 10); //Get numbers from 1 - 10}
    
@foreach(var number in numbers){
    @(number * 10)
}

如果要包含包含 @ 符号（例如电子邮件地址）的静态内容，则需要告诉 Razor 这个 @ 符号的实例不表示代码 . 您可以通过转义 @ 符号来做到这一点： 

For support, contact support@@domain.com

如果您想将内联变量与要在代码块中逐字呈现（逐字字符串）的项目混合在一起，可以通过三种方法告诉 Razor 代码暂停的位置以及文字文本或标记的开始位置。 第一个，如果附加项仅为文本，则在行中文本的第一个实例之前以 @: 为文本前缀： 

@{ var numbers = Enumerable.Range(1, 10); //Get numbers from 1 - 10 }

@foreach(var number in numbers){
    @(number * 10)@:&nbsp;
}

您只需要每行使用一次 @: 运算符：

@{var numbers = Enumerable.Range(1, 10); //Get numbers from 1 - 10 }
    
@foreach(var number in numbers){
    @(number * 10)@: * 10 = @(number * 10)
}

Razor 还会查找 html 标签。 如果它看到一个，它会跳出代码，只有在看到匹配的结束标记时才会跳回代码： 

@{ var numbers = Enumerable.Range(1, 10); //Get numbers from 1 - 10 }
    
@foreach(var number in numbers){
    <span>@(number * 10)&nbsp;</span>
}

Razor 可以识别自闭合标签：

@{ var numbers = Enumerable.Range(1, 10); //Get numbers from 1 - 10 }

@foreach(var number in numbers){
    @(number * 10)<br />
}

如果不想渲染 html 标签，可以使用 <text> 标签告诉 Razor 代码在哪里结束和重新开始。 <text> 标签不会呈现在浏览器中： 

@{ var numbers = Enumerable.Range(1, 10); //Get numbers from 1 - 10 }
    
@foreach(var number in numbers){
    <text>@(number * 10) * 10 = @(number * 10)&nbsp;</text>
}

代码块中的注释可以用两个正斜杠 // 表示，如前面所有示例中所示。 或者，您可以使用 /*...*/ 或 @*...*@。 如果要将服务器端注释放在代码块之外，请在 @* 行之前 

@{
    /*Get numbers from 1 - 10*/
    //Get numbers  between 1 - 10
    @*Get numbers between 1 - 10*@
    var numbers = Enumerable.Range(1, 10); 
}
    
@foreach(var number in numbers){
    @number
}

您可以使用 @*...*@ 语法在代码块之外添加注释： 

@* Get numbers between 1 - 10 *@

@{ var numbers = Enumerable.Range(1, 10); }

@foreach(var number in numbers){
    @number
}

注释不会呈现在浏览器中。
