什么是 ASP.NET Core？

ASP.NET Core 是用于构建基于 Web 的应用程序的 Microsoft 框架的最新版本。 它位于 .NET Core 之上，.NET Core 是一个开放源代码开发平台，由一组框架库，一个软件开发工具包（SDK）和一个运行时组成。

为什么要选择 ASP.NET Core？

Web 开发正在发生变革。越来越需要更多的模块化框架，您可以在其中确定要包含在应用程序中的功能。 应用程序应为云就绪的-设计为可以在任何平台上运行并可以快速扩展。 他们还应该采用客户端框架，并使开发 RESTful API 变得容易。 框架本身需要灵活。 它们需要快速迭代以提供新功能，以响应 Web 开发领域的新创新。 开发人员希望能够选择用于创作网站的工具。

一些框架已经解决了这些问题，例如在 Node.js 上运行的 Express。 没有旧版本的 ASP.NET 。 它已与仅 Windows 的 Web 服务器 Internet Information Services 结合在一起。 由于框架与完整的 .NET 框架绑定的方式，新功能需要花费很长时间才能上市。 它非常依赖 Visual Studio - 仅在 Windows 上运行的 IDE 的庞然大物。

请注意，称为 Visual Studio for Mac 的产品实际上并不是大多数 .NET 开发人员都熟悉的 IDE 版本。 它是支持 .NET Core 开发的 Xamarin Studio 版本。

ASP.NET 是以模块化设计为主的。HTTP管道由可以根据需要插入的独立组件组成。 这种方法带来的好处包括：

  • 您的应用程序更轻巧，因为它仅包含所需的组件
  • 您可以从多个来源中选择组件
  • 您可以选择在任意平台上托管您的应用程序
  • 与以前版本的ASP.NET相比，新功能的添加速度要快得多。

ASP.NET Core 是一种基于模型-视图-控制器（MVC）模式的Web开发框架。 最重要的是 Razor Pages 框架，面向更喜欢或更熟悉以页面为中心来构建Web应用程序的开发人员。 ASP.NET Core 还包括用于开发基于REST的Web服务（Web API）的框架。 包括基于 Web Sockets 的框架（SignalR）在内的工作也正在进行中，该框架能支持服务器启动的页面内容的实时更新。
