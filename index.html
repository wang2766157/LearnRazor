欢迎学习 Razor Pages

该站点专用于帮助希望使用ASP.NET Core Razor Pages Web开发框架来构建Web应用程序的开发人员。

什么是 Razor Pages？

ASP.NET Core Razor Pages 是一个以页面为中心的框架，用于构建动态的，数据驱动的网站，并清晰地关注各个方面。 Razor Pages 基于 Microsoft 的最新版本的 ASP.NET- ASP.NET Core，支持跨平台开发，可以部署到 Windows，Unix 和 Mac 操作系统。

Razor Pages 框架轻巧且非常灵活。 它为开发人员提供了对渲染 HTML 的完全控制。 该框架基于 ASP.NET Core MVC 构建，并且在 .NET Core 应用程序中启用 MVC 时默认情况下启用。 Razor Pages 是 .NET Core 上跨平台服务器端 HTML 生成的推荐框架。 您无需具有任何 MVC 知识或了解即可使用 Razor Pages。

Razor Pages 利用流行的 C＃ 编程语言进行服务器端编程，并使用易于学习的 Razor 模板语法 将 C＃嵌入 HTML 标记中，从而为浏览器动态生成内容。

谁应该使用 Razor Pages?

Razor Pages 适用于从初学者到企业级的各种开发人员。 它基于以页面为中心的开发模型，使 Web 开发人员熟悉其他以页面为中心的框架，例如 PHP，Classic ASP，Java Server Pages，ASP.NET Web Pages 和 ASP.NET Web Forms。 对于初学者来说，它也相对容易学习，它包含 ASP.NET Core 的所有高级功能，使其同样适用于大型、可伸缩的基于团队的项目。

如何获取

从2.0版开始，Razor Pages 包含在 .NET Core 中，可以作为 SDK（软件开发工具包）或运行时 免费下载。 该 SDK 包含用于创建 .NET Core 应用程序的运行时和命令行工具。 当您安装 Visual Studio 2017 Update 3 或更高版本时，将为您安装 SDK。 运行时用于运行 .NET Core 应用程序。 仅运行时安装旨在用于不进行开发的机器上。

为什么要使用 Razor Pages?

如果您想要一个动态网站（即定期添加内容的网站），则可以使用许多选项。您可以使用内容管理系统（CMS），其中有很多可供选择，包括 WordPress，Joomla！，Drupal，Orchard CMS 等。或者，您可以雇用某人为您建立合适的网站。或者，如果您有兴趣并有一定的编程天赋，则可以自己构建。

如果选择自己构建，则可以从多种编程语言和框架中进行选择。如果您是初学者，则可能要从易于学习，良好支持和强大的框架和语言入手。如果您正在考虑从事程序员职业，那么您可能想知道，在学习新框架时获得的技能将增强您对潜在雇主的价值。在这两种情况下，将C＃作为一种语言来学习并将 ASP.NET Core 作为一种框架来学习将使这些框打勾。如果您是一位经验丰富的开发人员，那么 Razor Pages 框架可能会以最少的工作量增加您的技能。
